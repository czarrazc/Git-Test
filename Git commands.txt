git add .
: 현재 디렉터리(.)와 그 하위 모든 변경 파일을 Git 스테이징 영역(index)에 추가

git branch
: 현재 존재하는 브랜치 목록 보기

git branch <브랜치이름>
: 단순히 브랜치만 생성함 (자동 전환은 안 됨)
 현재 HEAD가 가리키는 커밋에서 브랜치가 생성됨

git branch -m 새이름
: 현재 브랜치 이름 변경

git push <원격이름> <브랜치이름>
: <브랜 브랜치에서 커밋한 내용들을 원격의 main 브랜치에 반영합니다.

git switch <이동할 브랜치 이름>
: 브랜치간 전환(이동)
